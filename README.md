# Proiect_SSD_books
**Description of the web app**:
Web application in which the users can search for books or movies to their watch or read list , can share and mark their progress and can review a book or a movie .

# Use cases:
***Use case 1 :***
Primary Actor:User
Goal:Find movies/books 

Interacting with the System:

1.User enters query and submits.

2.System calls OMDb/Google Books.

3.System shows paginated results with title, year, cover.

***Use case 2 :***
Primary Actor: User
Goal: Inspect full details of a selected title.

Interacting with the System:

1.The user clicks on a movie or book from the list.

2.The app shows details such as the title, description, author/director, year, and rating.
